## 🎯 推荐学习路径

### 路径 1: 论文研究工作流（1-2 周）

**目标**: 掌握完整的论文研究流程

```
Day 1-3: paper-analysis-详解.md
  └─ 学习 6 维度论文分析框架
  └─ 实践: 分析 2-3 篇经典论文

Day 4-7: research-analyst-详解.md
  └─ 学习系统化文献综述
  └─ 实践: 做一个小领域的调研

Day 8-10: deep-research-详解.md
  └─ 学习多 Agent 并行调研
  └─ 实践: 调研 Transformer 进展

Day 11-14: 综合项目
  └─ 选择研究方向
  └─ 完整调研 + 论文分析
  └─ 输出研究报告
```

**产出**:
- ✅ 3-5 篇论文深度分析
- ✅ 1 份系统化文献综述
- ✅ 1 个研究方向报告
