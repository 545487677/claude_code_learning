# 学术论文分析 Skill

基于第一性原理的系统化学术论文分析工具。

## 功能特性

- ✅ **6 维度深度分析**：Task、Challenge、Insight & Novelty、Potential Flaw、Motivation、关键公式
- ✅ **第一性原理思考**：从问题本质出发推演思路
- ✅ **结构化输出**：清晰的 markdown 格式
- ✅ **研究笔记生成**：个人思考、相关工作、未来方向
- ✅ **结果保存**：自动保存为 markdown 文件

## 快速开始

### 基础使用

```
分析这篇论文：

[粘贴论文内容]
```

### 保存结果


## 分析维度

### 1. Task（问题形式化）
- 输入输出定义
- 数学形式化
- 约束条件
- 问题类型

### 2. Challenge（传统方法困境）
- 主要挑战识别
- 原因分析
- 影响评估

### 3. Insight & Novelty
- **Inspiration**：启发来源
- **Insight**：核心洞察
- **Novelty**：创新点分解（问题→启发→设计）

### 4. Potential Flaw
- 情境局限性
- 数据敏感度
- 研究价值评估

### 5. Motivation
- 第一性原理推演
- 问句形式思考链

### 6. 关键公式
- 核心数学表达
- 文本形式（无 LaTeX）

## 触发方式

以下任何表达都可以触发：
- "分析这篇论文"
- "paper analysis"
- "analyze this paper"
- "深度分析"
- "论文分析"

## 输出格式

默认输出 markdown，包含：
- 6 个维度的结构化分析
- 公式用文本形式
- 清晰的层级结构
- 无客套话，直接输出结果

额外生成：

## 注意事项

- 提供完整论文内容效果最佳
- 部分内容（如仅摘要）会标注局限性
- 保持批判性思维，客观分析
- 公式自动转换为文本形式

## 反馈与改进

如果分析不符合预期，可以：
1. 提供更多论文细节
2. 明确关注的维度
3. 指定输出格式要求
